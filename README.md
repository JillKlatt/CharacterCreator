# DM NOTEBOOK

 
## Basic Character, Campaign, and Weapon Creation at Your Fingertips! 

 
### Install Instructions:
Fork and clone this repo
<br />
Run ```bundle install```
<br />
Run ```rails s``` and use localhost:3000 on your browser
